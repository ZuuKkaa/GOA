GitHub არის ვებ პლათფორმა,GitHub ს დეველოპერები იყენებ რომ ატვირთონ თავიანთი პროექტები და განავითარონ სხვა დეველოპერები თავისი კოდით.
Git გვეხმარება რომ ავტვირთოთ დავალებები და პროექტები.დეველოპერები უზიარებდენ აქ თავის ნამუშევრებს და პროექტებს ერთმანეთს.



Git არის სისტემა რომელიც დეველოპერებს ეხმარება თავისი ნამუშევრების ატვირთვაში გითჰაბზე.
Git ი საშვალებას აძლევს დეველოპერებს ითანამშრომლონ და გააზიარონ თავიანთი პროექტები.
